# Memoria del Proyecto Silhouette Web

Este archivo contiene hechos clave, decisiones importantes y contexto persistente específico para el proyecto Silhouette Web. Su propósito es mantener la continuidad del contexto a través de diferentes sesiones o instancias del Agente CLI.

---

**Directiva para el Agente CLI:**

Al inicio de cada sesión, o cuando se te indique cargar el contexto del proyecto, por favor, sigue estos pasos:

1.  **Lee el `MAPA_DE_MD.md`** para identificar todos los archivos Markdown relevantes del proyecto.
2.  **Procesa los archivos Markdown de forma inteligente:**
    *   Prioriza la lectura completa de los archivos listados bajo "Archivos de Contexto Operacional" y "Archivos de Contexto Estratégico y Arquitectónico" en `MAPA_DE_MD.md`.
    *   Para archivos extensos o de documentación general, enfócate en extraer los puntos clave, resúmenes y cualquier información que parezca relevante para el estado actual del proyecto o los planes de acción pendientes. Evita la sobresaturación de información.
    *   Si encuentras directivas específicas para tu operación (como esta), incorpóralas a tu contexto de sesión.
3.  **Incorpora los hechos de este archivo (`PROJECT_MEMORY.md`)** a tu memoria operativa para la sesión actual.

---

## Hechos Clave del Proyecto

*   **Objetivo Principal:** Replicar el diseño del sitio `https://albertofarah.com/` utilizando React y tecnologías modernas, preparado para integración de IA.
*   **Estado:** Inicio desde cero. Workspace limpio.
*   **Arquitectura:** React para frontend, preparado para self-hosting (Docker).
*   **Requisitos Especiales:**
    *   Replicación visual exacta.
    *   Animaciones sincronizadas con scroll.
    *   Futura integración de Agente IA "omnipotente".
*   **Directiva de Estilo:** Premium, moderno, usar animaciones ricas.
